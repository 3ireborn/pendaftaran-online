Cara Kerja

  1. Pengunjung klik "Daftar Sekarang" → buka form pendaftaran (file terpisah).
  2. Form mengikuti SOP perusahaan (POM) — berisi Nama Ibu Kandung dan semua field wajib.
  3. Leader menerima notifikasi WA sesuai nomor di data leaders.json, lalu input manual ke sistem perusahaan bila closing.

Catatan: bila leader ingin otomatis follow-up via admin pusat, buka halaman admin (file admin terpisah) dan isi token FonNTE + GitHub repo untuk sinkronisasi.
